== Description ==

Widget which displays latest posts including custom posttypes. You can set the number of posts to be displayed.
Just drag in the widget and done... your most recent posts of all types are displayed.

== Installation ==

1. Upload `simple-latest-posts-including-custom-posttypes.zip` to the `/wp-content/plugins/` directory and unzip
1. Activate the plugin through the 'Plugins' menu in WordPress
